Structure d'un projet

  • app/ - Fichiers de configurations et templates
  • bin/ - Fichiers binaires et executables
  • src/ - Votre code, vos bundles !
  • tests/
  • var/ - Contient le cache, les logs ainsi que les sessions
  • vendor/ - Contient toutes les librairies PHP
  • web/ - Dossier racine de la vhost, point d'entrée de l'application

Utilisation de la console


Ouvrir un terminal/invite de commande et entrer



                        php bin/console

Création d'un vhost


                        php bin/console server:run

Création d'un bundle


                        php bin/console generate:bundle

Création d'une Entity


                        php bin/console doctrine:generate:entity

Mise à jour de la base de données


                        php bin/console doctrine:schema:update --dump-sql
					    

                        php bin/console doctrine:schema:update --force

Génération du CRUD


                        php bin/console doctrine:generate:crud
